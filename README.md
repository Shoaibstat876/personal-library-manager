# Personal Library Manager

Personal Library Manager is a Python command-line application for managing a personal book collection.

The app allows users to add, remove, search, display, and track books from a simple terminal-based interface. It also includes basic reading statistics and save/load functionality so the library data can remain available after the program closes.

## Project Overview

This project was created to practice Python fundamentals through a practical command-line application.

The main goal was to build a simple book management system using core Python concepts such as user input, lists/dictionaries, functions, condition handling, and file handling.

## Features

* Add a book with title, author, year, genre, and read/unread status
* Remove a book from the library
* Search books by title or author
* Display all saved books
* Show reading statistics
* Track total books and percentage of books read
* Save and load library data
* Simple terminal-based user interface
